# Descriptive Statistics & Univariate Analysis

## Project Overview
This project focuses on performing Exploratory Data Analysis (EDA), SQL-based business analysis, and multivariate analysis to extract meaningful business insights from the dataset.

The objective is to analyze trends, patterns, customer behavior, and business performance using statistical analysis, SQL queries, and visualizations.

The project also includes designing a static dashboard mock-up to highlight key business KPIs.

---

# Objectives

- Perform descriptive statistical analysis
- Conduct univariate and multivariate analysis
- Answer business-driven questions using SQL
- Explore relationships between variables
- Create advanced visualizations
- Generate actionable business insights
- Design a KPI dashboard mock-up

---

# Business Questions

The following business questions were analyzed using SQL queries:

1. What are the top 5 products by revenue in the last 6 months?
2. What is the monthly sales trend?
3. Which customer segment contributes the highest revenue?
4. What are the top-performing regions based on sales?
5. Which products have the highest return rates?
6. What is the customer acquisition trend over time?
7. Which marketing channels generate the most conversions?

---

# SQL Analysis

SQL queries were written and executed to answer the business questions.

## SQL Concepts Used
- Filtering (`WHERE`)
- Aggregation (`SUM`, `AVG`, `COUNT`)
- Grouping (`GROUP BY`)
- Sorting (`ORDER BY`)
- Date Functions
- Multi-table Joins
- Subqueries

The queries helped identify trends, high-performing products, customer behavior patterns, and operational insights.

# Descriptive Statistics

Statistical analysis was performed on numerical and categorical variables.

## Analysis Included
- Mean
- Median
- Mode
- Standard Deviation
- Variance

- Range Analysis

This helped summarize the dataset and identify unusual patterns or anomalies.

Univariate analysis was conducted to understand the distribution of individual variables.

# Multivariate Analysis & Correlation

Advanced visualizations were created to analyze relationships between multiple variables.

## Visualizations Used
- Scatter Plots
- Heatmaps
- Pair Plots
- Correlation Matrix

## Relationships Explored
- Marketing Spend vs Sales
- Customer Age vs Product Category
- Website Traffic vs Conversion Rate
- Discounts vs Revenue

These analyses helped uncover patterns, dependencies, and correlations between business metrics.
